# Concept Analyze — 概念关联发掘

## 使用方式

```
/concept-analyze
```

默认增量模式：只处理上次分析后新增的概念。每季度可全量重跑一次重建基准。

---

## 执行步骤

### Step 1 — 读取缓存，确定模式

读取 `memory/concept_relations.md`：

- **文件存在**：提取 `last_analysis` 日期和 `concept_count`，进入增量模式
- **文件不存在**：全量分析模式（首次运行）

向用户确认模式：
> 缓存时间：{last_analysis}，上次记录 {concept_count} 个概念。当前为增量模式，只处理新增概念。继续？

等待确认。若用户说「全量」或「重建」，切换到全量模式。

---

### Step 2 — Pass 1：frontmatter 扫描（极省 token）

**增量模式**：只处理 `date > last_analysis` 的概念页

```bash
cd 概念页/
for f in *.md; do
  [[ "$f" == "INDEX.md" ]] && continue
  awk '/^---/{c++;next} c==1{print}' "$f" | head -8
done
```

从每个概念提取：`name` / `domain` / `date` / `source` / `tags`

**数据质量检查（每页必查）：**
- frontmatter 是否完整（6 个字段：name / domain / date / source / tags，无禁用字段）
- `domain` 值是否在 10 个固定域内（哲学/心理学/经济学/社会学/传播学/管理学/生物学/人类学/政治学/艺术）
- `pattern/` 值是否在 8 个固定值内
- `apply/` 值是否在 10 个固定值内（无谈判）
- `discipline/` 值是否在白名单内（35个）

**全量模式**：所有概念页执行上述操作。

---

### Step 3 — 集群归属分析

对每个新概念，通过 domain + tags 的重叠度，判断归属：

**归属规则（按优先级）：**
1. 与某集群内 ≥2 个概念共享相同 `apply/` + `pattern/` → 归入该集群
2. 与某集群内 ≥1 个概念共享相同 `discipline/` → 候选归入
3. 无明显归属 → 标记为孤立概念

**同时检查：**
- 新概念是否与现有概念名高度相似（潜在重复）
- 新增孤立概念有哪些

---

### Step 4 — Pass 2：全文扫描（仅候选 pair）

对 Pass 1 中发现的"可能强关联但尚未链接"的概念对，读取全文：

```bash
cat 概念页/{概念名}.md
```

验证：
- 是否已有 `[[对方概念名]]` 的出链
- 正文中是否自然涉及对方概念的核心机制

只对有价值的 pair 执行此操作（通常 ≤10 对），不读整个库。

---

### Step 5 — 展示分析结果

输出格式：

```
【/concept-analyze 分析结果】

新增概念：N 个（自 {last_analysis} 起）

▸ 新归入集群
  - {概念名} → 集群 {X}（共享 {标签}）

▸ 新增孤立概念
  - {概念名}

▸ 建议添加的 [[]] 链接
  - {概念A} → 建议链接 [[{概念B}]]，理由：{一句话}

▸ 数据质量问题
  - {概念名}：{问题描述}

▸ 待处理（已有问题未变化）
[列出 concept_relations.md 中未解决的问题]
```

---

### Step 6 — 更新缓存

用 Write 工具覆写 `memory/concept_relations.md`，更新：
- `last_analysis`：今天日期（`date` 命令获取）
- `concept_count`：当前总数
- `orphan_count`：当前孤立数
- 集群：合并新成员
- 孤立列表：移除已有出链的概念，添加新孤立概念
- 待处理：更新已解决 / 新发现的问题

---

### Step 7 — 可选：批量添加链接

询问是否立即为本次分析中"建议添加 [[]] 链接"的 pair 执行写入：

> 发现 N 个建议链接。要批量写入吗？（全部 / 选择 / 跳过）

若用户选择写入：逐个用 Edit 工具在正文中自然插入 `[[概念名]]`，不改动原有句子结构，只在已有提及处加链接符号。

---

## 全量模式附加步骤

全量模式在 Step 2 完成后，额外执行：

```bash
# 找出所有有出链的概念页
grep -l "\[\[" 概念页/*.md

# 对比全部概念列表，得出孤立概念
```

重建完整集群图谱，不依赖旧缓存。
