# 叙事工程：Douglas 方法论提取

> Reference: Pamela Douglas, *Writing the TV Drama Series* (USC 电影艺术学院教授)
> 提取目的：将电视剧编剧的工业级 craft 方法论转化为恐怖叙事的操作工具。不引用原文，只保留经恐怖迁移后的可用公理。

---

## 一、硬停点网格

任何篇幅的叙事都可以按等间距切分，每个切口必须是一个**张力节点**——不是休息位，是爆发口。

**节点设计清单（任选其一或组合）：**
1. 信息揭露 — 观众刚知道了一件可怕的事
2. 地理封锁 — 安全空间缩小了
3. 盟友丧失 — 可信任的人/物减少了一个
4. 认知翻转 — 之前确信的事被推翻
5. 时间压力 — 倒计时开始或加速
6. 身体代价 — 主角付出了可见的损失

**尺度适配：**
| 篇幅 | 切分单元 | 节点数 |
|------|---------|-------|
| 长片/长篇 | 幕(20-30min) | 4-5个 |
| 短片(10-30min) | 段(5-8min) | 3-4个 |
| **5分钟短片** | **节拍(0.5-1.5min)** | **3-5个** |

---

## 二、Beat（戏剧节拍）

最小不可分叙事单元。一个 beat = 一个触发事件 + 角色反应 + 意义转变。

**检查清单（每个 beat 必须全过）：**
- [ ] 有明确触发事件（非过渡性描写）
- [ ] 角色做出反应（非被动承受）
- [ ] 结束状态 ≠ 开始状态（必须有转变）
- [ ] 推动了下一个 beat

**5分钟短片的 beat 模板（3-beat 精简版）：**

| Beat | 时间 | 功能 |
|------|------|------|
| B1 | 0-1.5min | 正常 → 不对劲（埋第一个裂痕） |
| B2 | 1.5-3.5min | 不对劲 → 恐惧确认（升级+合理化失败） |
| B3 | 3.5-5min | 恐惧确认 → 更深的恐惧 / 反转 |

5-beat 版本在 B2 和 B3 之间插入「认知翻转」和「绝望积累」。

## 三、信息不对称四种模式

紧张感来自信息的不对称分配，不只是来自危险本身。不同篇幅的混用策略不同：

| 模式 | 定义 | 长篇 | **5分钟短片** |
|------|------|------|-------------|
| **Dread** | 观众知道 > 角色不知道 | 可做主线 | ✅ 主打，打透 |
| **Curiosity** | 角色 > 观众 / 叙述者不可靠 | 做副线或主线 | ✅ 主打，打透 |
| **Surprise** | 双方都不知道 | 用于高潮 | ✅ 主打，打透 |
| **Helplessness** | 都知道但无能为力 | 用于终局 | ✅ 主打，打透 |

**5分钟规则：只选一种模式主打。** 没有空间切换情绪状态。

### Dread（观众先行）
- 开头给观众一个角色没注意到的信号
- 角色的每个"正常行为"在观众眼里都变紧张
- 高潮 = 角色终于看到了观众一直看到的东西
- 适合：鬼片、跟踪、入侵类

### Curiosity（角色先行）
- 通过角色视角呈现世界，但世界有"毛边"
- 观众逐渐意识到角色的感知有问题
- 高潮 = 观众理解与角色体验断裂
- 适合：心理恐怖、不可靠叙述者

### Surprise（双方无知）
- 先建正常情境，用节奏让观众放松
- 揭露快且狠，不给铺垫时间
- 适合：jump scare 密集型、反转恐怖

### Helplessness（双方知情）
- 不藏信息，恐惧来自必然性
- 节奏重点在逼近而非揭露
- 适合：宇宙恐怖、宿命论、封闭空间

## 四、人物向度（纵向发展）

人物不需要"成长弧线"。当没有横向发展空间时，用纵向发展替代：

> 横向 = 朝目标推进（情节驱动）→ 短篇可能不够用
> 纵向 = 内心冲突逐层显露（张力驱动）→ 短篇的核心操作

**5分钟里的人物处理三选一：**

| 方案 | 操作 | 适用 |
|------|------|------|
| **暗示法** | 一句话/一个动作暗示创伤或弱点，不展开 | 心理恐怖 |
| **功能化** | 人物是恐惧感受器，不需要 backstory | jump scare / 宇宙恐怖 |
| **极简弧线** | 前80%状态A → 一个事件 → 后20%状态B | 需要人物转变时 |

## 五、主角驱动原则

恐惧感来自主角的选择和无力感，不是怪物的炫技。

```
错误：怪物计划X → 执行 → 主角逃跑 → 追逐 → 对决
正确：主角弱点Y → 陷入困境Z → 怪物是压力放大器 → 主角的 choices 导致结果
```

怪物的可怕程度 = 主角无力感的函数。主角的个人弱点（恐高症、幽闭恐惧、创伤）应该成为剧情引擎。

## 六、Teaser 即全片（5分钟专用）

整部片子就是一个 Teaser。三条铁律：

**1. 前15-20秒建立不安感**
- ❌ 不解释背景、不展示环境、不独白设定
- ✅ 直接扔进已经不对劲的情境
- ✅ 用一个感官细节定调（声音/光线/温度）

**2. 调性从头到尾不摇摆**
- 5分钟没空间做反差（除非精确计算在认知翻转处）

**3. 结尾让问题更重，不更轻**

五种结尾策略：
| 策略 | 操作 | 特点 |
|------|------|------|
| 开放悬置 | 未解决，黑屏 | 最通用 |
| 反转颠覆 | 最后推翻一切 | 高风险高回报 |
| 循环回归 | 回到开头但意味不同 | 宿命感 |
| 渐弱消逝 | 恐惧不爆发而是淡出 | 不安残留 |
| 戛然而止 | 最高潮处切断 | 最暴力 |

## 七、节奏网格（5分钟专用）

```
时间:   0:00   1:00   2:00   3:00   4:00   5:00
        │      │      │      │      │      │
张力:   ━━╱╲━━╱╲━━╱╱╱╱╱╱╱╱╱╱╱╱╱
        建    置   升    级   高    潮

✓ 0:15  — 是否已建立不安？
✓ 1:00  — 第一个beat是否完成？有无意义转变？
✓ 2:30  — 中段是否已升级？有没有原地打转？
✓ 3:30  — 是否已发生认知翻转？（没有=太平）
✓ 4:30  — 绝望感/张力是否达峰值？
✓ 5:00  — 结尾是否是硬停点？
```

## 八、失败模式速查

| 失败表现 | 根因 | 修法 |
|---------|------|------|
| 前45秒还在介绍角色/环境 | 违反Teaser原则1 | 删掉开头，从第一个不对劲的时刻开始写 |
| 中段没有实质推进 | 某个beat缺意义转变 | 合并或删掉那个beat |
| 塞入了背景故事/多角色关系 | 5分钟不需要B故事 | 砍掉或压缩成一颗粒（一句话/一个物件） |
| 前半段 dread 后半段 surprise | 信息不对称混用 | 全片只选一种模式 |
| 结尾解释了所有悬念 | 违反Teaser原则3 | 删掉解释，换成硬停点 |
| jump scare 满天飞/角色尖叫标配 | 缺认知机制支撑 | 每个恐惧节点回查它在操控什么认知机制 |

## 九、生成流程（嵌入恐怖大师 Skill）

### 长篇/短片（≥10分钟）

```
Step 1: 硬停点网格 → 选切分单元+节点类型
Step 2: 信息不对称 → 选主导模式（可混用2种）
Step 3: 人物向度 → 设计纵向发展路径
Step 4: 主角驱动检验 → 怪物是否喧宾夺主？
Step 5: Beat 展开 → 逐节拍填充
Step 6: 自检 → 过失败模式表
```

### 5分钟短片

```
Step 1: 信息不对称选择器 → 四选一，唯一主打
Step 2: Beat 设计 → 3或5个beat，全过检查清单
Step 3: Teaser 检验 → 前15-20秒不安感确认
Step 4: 节奏网格填充 → 时间轴标注
Step 5: 结尾策略 → 五选一
Step 6: 自检 → 过失败模式表
```
