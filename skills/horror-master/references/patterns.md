# 恐怖大师 · 编剧套路（路由索引）

本文件是索引。具体内容在 `patterns/` 子目录的独立模块文件中，按需调用。

---

## 模块地图

| 文件 | 内容 | 什么时候用 | 行数 |
|------|------|-----------|------|
| **[pacing.md](patterns/pacing.md)** | 叙事节奏、张力工程、硬停点网格、节奏形态 | 设计整体结构 / 控制节奏 / 检验张力分布 | ~70 |
| **[information.md](patterns/information.md)** | 知情权操作工具箱、四种基础位置、动力学模式、**反转类型目录** | 设计信息差 / 选定反转机制 / 5分钟短片核心决策 | ~230 |

| **[elements.md](patterns/elements.md)** | 空间/身体/社会结构/信息感知/时间结构的可借用元素 | 找场景灵感 / 选身体恐怖类型 / 设计社会背景 | ~60 |
| **[subgenres.md](patterns/subgenres.md)** | 怪物片vs鬼片、社会/家族/民俗恐怖的结构差异 | 确定子类型方向 / 检查是否符合类型惯例 | ~60 |
| **[tropes-anti.md](patterns/tropes-anti.md)** | 409条Tropes速查（按脉络分组）、反套路分析、元叙事策略 | 查某个trope是否被用烂了 / 找反套路灵感 / 元叙事设计 | ~80 |

## 快速导航

### 写作流程中的调用顺序

```
1. 定子类型 → subgenres.md（确认类型框架）
2. 设计角色 → character.md（主角+威胁）
3. 设计信息操作 → information.md（知情权指纹 + 反转类型）
4. 设计节奏 → pacing.md（停点 + 形态）
5. 找元素 → elements.md（场景/身体/社会填充）
6. 自检 → craft-checklist.md
7. 避俗 → tropes-anti.md（检查有没有踩坑）
```

### 按问题找模块

| 你在问… | 去… |
|---------|-----|
| "这个故事的节奏对不对？" | pacing.md |
| "观众应该比角色多知道什么？" | information.md → 基础位置 |
| "反转怎么设计？" | information.md → 反转类型目录 |
| "主角怕什么？威胁是什么？" | character.md |
| "开头怎么写？" | character.md → 开篇 |
| "结尾怎么收？" | character.md → 结尾 |
| "场景放在哪？" | elements.md → 空间 |
| "这种恐怖属于哪个子类型？" | subgenres.md |
| "这个套路是不是被用烂了？" | tropes-anti.md |
| "我想反套路" | tropes-anti.md → 反套路与元叙事 |

---

*套路不是教条——每条套路都有它有效的认知机制，理解了机制才能决定什么时候该用、什么时候该破。*
