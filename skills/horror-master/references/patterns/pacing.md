# 结构性套路 · 叙事节奏与张力工程

## 「日常→裂痕→崩塌」三段式
- **操作**：前1/3建立日常秩序（让观众锚定"正常"），中1/3引入异常（日常开始出现裂痕），后1/3全面崩塌（异常吞噬正常）
- **概念脉络**：不安生成线——阴森感的"熟悉变陌生"需要先建立足够的"熟悉"；混沌边缘——系统从有序到临界的相变过程
- **经典用例**：《罗斯玛丽的婴儿》（新婚日常→邻居越来越怪→撒旦教真相）、《遗传厄度》（家庭哀悼→诡异事件→邪教降临）
- **变体**：日常可以极短（开头5分钟的震撼死亡→回溯日常→再崩塌），如《惊声尖叫》开头

## 「越解释越不安」渐进披露
- **操作**：每次给出一点新信息，看似在解释，实际让恐惧更深——每一条新信息都在扩展威胁的范围
- **概念脉络**：测不准与观察者效应——每一次"确认"都制造更多不确定性；悬念与惊吓——信息分配控制恐惧节奏
- **经典用例**：《怪形》（每次检测看似缩小了嫌疑范围，实际让你怀疑更多人）
- **编剧要点**：新信息必须同时解决一个旧问题和制造一个新问题。如果只解决不制造，恐惧衰减；如果只制造不解决，观众困惑

## 「不可能的逃出」封闭空间
- **操作**：角色被困在一个无法离开的空间——物理封锁（暴风雪/荒岛/密室）或心理封锁（诅咒/附身/必须留下）
- **概念脉络**：阈限性——空间从"过渡地带"变成"监狱"；边界侵犯——安全边界被不可逆地穿透
- **经典用例**：《闪灵》（暴风雪封闭酒店）、《异形》（太空船无处可逃）、《寂静之地》（出声就死）
- **变体**：封锁可以逐步收紧——先是"可以走但不想走"，再是"想走但走不了"，最后是"走得了但外面更危险"

## 硬停点网格（工程方法）

任何篇幅都可以按等间距切分。每个切口是一个**张力检验点**——到这里时问自己：观众还在吗？

**节点类型启发（不是清单）：**

空间类：地理封锁 / 出路消失 / 边界渗透
信息类：信息揭露 / 认知翻转 / 信息污染
关系类：盟友丧失 / 孤立加剧 / 敌友难辨
时间类：时间压力 / 回旋余地消失 / 过去追上
身体类：身体代价 / 感官失效 / 控制感丧失
心理类：理性防线破裂 / 参照系崩塌 / 自我怀疑

使用方式：每个停点选1-2种。同一类型可重复（每次升级一层）。可发明列表外的新类型。

**尺度参考（不是规定）：**
| 篇幅 | 切分粒度 | 大约几个停点 |
|------|---------|-------------|
| 长片/长篇 | 幕(20-30min) | 4-6 |
| 短片(10-30min) | 段(5-8min) | 3-4 |
| 5分钟短片 | 节拍(0.5-1.5min) | 3-5 |

## 节奏形态（工程方法）

张力曲线不固定。可选方向：

单调上升（持续爬升无喘息）→ 压抑型
锯齿波（升→释→升，释放不完全）→ jump scare 密集型
阶梯式（升→平台→更高升）→ 心理恐怖
假释（看似安全其实没有）→ 虚假安全感型
延迟爆发（长时间低张力后一次释放）→ 氛围/宇宙恐怖
碎片化（大量短促冲击）→ 实验性

随时自问：现在张力偏高还是偏低？上次释放是什么时候？下一个节拍的方向对吗？
